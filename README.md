# Notas Prácticas

Este proyecto es un sitio web estático diseñado como un cuaderno de notas para una materia de `Prácticas Profesionalizantes I` de la tecnicatura en Análisis de Sistemas. Está pensado para ser un apoyo visual y práctico de conceptos básicos de HTML, CSS y algunas herramientas relacionadas.

## 🔍 ¿Qué incluye?

- `index.html`: página de bienvenida con navegación hacia las distintas secciones del proyecto.
- `html.html`: contenido teórico y ejemplos sobre HTML, incluyendo notas sobre etiquetas, tablas y caracteres especiales.
- `css.html`: introducción a CSS y ejemplos prácticos para aprender a dar estilo a una página.
- `horarios.html`: página con información de clases y horarios.
- `css/style.css`: estilos globales del sitio, diseño del menú y presentación general.
- `js/copiarCodigo.js`: función para copiar el código del ejemplo al portapapeles desde los bloques de código.

## 🎯 Características principales

- Diseño sencillo y moderno con colores suaves y tipografía legible.
- Menú de navegación con submenús desplegables en la página principal.
- Bloques de código con botón para copiar ejemplos fácilmente.
- Navegación entre páginas: `Inicio`, `HTML`, `CSS`, `Horarios`.
- Páginas construidas con HTML y CSS puros; no requiere servidor.

## 🚀 Cómo usar

1. Abre el proyecto en tu editor o directamente en el navegador.
2. Abre `index.html` para comenzar.
3. Navega por los enlaces para ver las notas de HTML, CSS y horarios.
4. En las páginas de contenido, prueba los botones de copia para llevar el código al portapapeles.

## 💡 Ideas de mejora

- Hacer que el menú funcione en móviles con `click` en lugar de solo `hover`.
- Añadir formularios o un menú fijo para mejorar la navegación en pantallas pequeñas.
- Ampliar el contenido con más ejemplos prácticos y ejercicios.

## 📌 Observaciones

- El sitio es completamente estático y no tiene backend.
- Las páginas `html.html` y `css.html` usan el mismo estilo compartido en `css/style.css`.
- El proyecto es ideal para practicar y aprender cómo crear una pequeña web educativa.
