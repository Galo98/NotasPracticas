¿Qué es un analista de sistemas?

Es el profesional encargado de relevar, interpretar, analizar y documentar las necesidades de un cliente o usuario para luego traducirlas en soluciones informáticas funcionales.

Basicamente:

  • Traduce las necesidades en soluciones.
  • Es el puente entre el usuario y el equipo técnico.

No tiene que ser un experto en programación, pero sí debe comprender cómo funciona un sistema informático.
