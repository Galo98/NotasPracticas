Introducción a CSS

¿Qué es CSS?

CSS (Cascading Style Sheets) es un lenguaje de hojas de estilo utilizado para describir la presentación de un documento HTML. Permite controlar el diseño, los colores, las fuentes y otros aspectos visuales de una página web.

Con CSS, puedes aplicar estilos a elementos HTML específicos o a grupos de elementos utilizando selectores. Puedes definir reglas de estilo que se aplican a diferentes partes de tu página web, lo que te permite crear diseños atractivos y personalizados.

Ejemplo de uso de CSS

A continuación se muestra un ejemplo básico de cómo se puede usar CSS para dar estilo a una página web:

¡Probalo en tu editor copiando el código! 
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Ejemplo de CSS</title>
    <style>
        body {
            font-family: Arial, sans-serif;
            background-color: #f0f0f0;
            color: #333;
        }
        h1 {
            color: #007BFF;
        }
        p {
            line-height: 1.5;
        }
        .resaltado {
            background-color: yellow;
        }
        .centrado {
            text-align: center;
        }
        .rojo {
            color: red;
        }
        .fondo-azul {
            background-color: blue;
            color: white;
        }
    </style>
</head>
<body>
    <h1 class="centrado">¡Hola, CSS!</h1>
    <p>Este es un ejemplo de cómo usar CSS para dar estilo a una página web.</p>
    <p class="resaltado">Este texto está resaltado.</p>
    <p class="rojo">Este texto es rojo.</p>
    <p class="fondo-azul">Este texto tiene fondo azul y texto blanco.</p>
</body>
</html>

En este ejemplo, se definen estilos para el cuerpo de la página, los encabezados, los párrafos y algunas clases específicas. Al aplicar estas reglas de estilo, se puede controlar la apariencia de la página web de manera efectiva.

Estilos CSS para tablas

Para darle estilo a una tabla, se pueden usar las propiedades CSS. A continuación se muestra un ejemplo de una tabla con estilos CSS:

En la tabla anterior se muestra una tabla con estilos CSS. La propiedad border se usa para definir el borde de la tabla y la propiedad padding se usa para definir el espacio entre el contenido de la celda y el borde de la celda.

Propiedad hover

La propiedad hover se usa para aplicar estilos a un elemento cuando el usuario pasa el mouse sobre él. A continuación se muestra un ejemplo de cómo usar la propiedad hover para cambiar el color de fondo de una celda de una tabla cuando el usuario pasa el mouse sobre ella:

¡Probalo en tu editor copiando el código! 
<table style="border: 1px solid black; border-collapse: collapse;">
    <tr>
        <th style="border: 1px solid black; padding: 5px;">Nombre</th>
        <th style="border: 1px solid black; padding: 5px;">Apellido</th>
        <th style="border: 1px solid black; padding: 5px;">Telefono</th>
    </tr>
    <tr>
        <td style="border: 1px solid black; padding: 5px;">Juan</td>
        <td style="border: 1px solid black; padding: 5px;">Navarro</td>
        <td style="border: 1px solid black; padding: 5px;">00000000</td>
    </tr>
    <tr>
        <td style="border: 1px solid black; padding: 5px;">Milagros</td>
        <td style="border: 1px solid black; padding: 5px;">Romero</td>
        <td style="border: 1px solid black; padding: 5px;">11111111</td>
    </tr>
</table>
<style>
    td:hover {
        background-color: yellow;
    }
</style>

El resultado de la tabla anterior se muestra a continuación:

Nombre Apellido Telefono
Juan Navarro 00000000
Milagros Romero 11111111

En el ejemplo anterior, se define un estilo para la propiedad hover que cambia el color de fondo de una celda a amarillo cuando el usuario pasa el mouse sobre ella. Esto permite resaltar la celda que el usuario está interactuando, mejorando la experiencia de usuario al navegar por la tabla.

Menú de navegación

Los menús de navegación permiten organizar los enlaces principales de una página web. Generalmente se utilizan para acceder rápidamente a distintas secciones del sitio.

En HTML, los menús suelen construirse utilizando listas. Cada opción del menú se representa con un elemento <li> y los enlaces se crean con la etiqueta <a>.

En el siguiente ejemplo se muestra cómo crear un menú horizontal simple utilizando HTML y CSS.

¡Probalo en tu editor copiando el código! 
<nav class="menu">

    <ul>

        <li>
            <a href="#">Inicio</a>
        </li>

        <li>
            <a href="#">Notas</a>
        </li>

        <li>
            <a href="#">Contacto</a>
        </li>

    </ul>

</nav>

<style>

    .menu ul {
        list-style: none;
        margin: 0;
        padding: 0;

        display: flex;

        background-color: #333;
    }

    .menu a {
        display: block;

        padding: 15px 20px;

        color: white;

        text-decoration: none;
    }

    .menu a:hover {
        background-color: #555;
    }

</style>

El resultado del menú anterior se muestra a continuación:

  • Inicio
  • Notas
  • Contacto

Paso 1: Crear la estructura HTML

La etiqueta <nav> se utiliza para representar una sección de navegación. Dentro de ella se agrega una lista utilizando <ul>. 

<nav class="menu">

    <ul>

        <li>
            <a href="#">Inicio</a>
        </li>

    </ul>

</nav>

Cada elemento <li> representa una opción del menú y cada etiqueta <a> representa un enlace.

Paso 2: Eliminar estilos automáticos

Las listas HTML tienen estilos predeterminados aplicados por el navegador, como márgenes, espacios internos y viñetas.

Para eliminarlos se utiliza: 

.menu ul {
    list-style: none;
    margin: 0;
    padding: 0;
}

La propiedad list-style: none elimina las viñetas de la lista.

Paso 3: Convertir el menú en horizontal

Por defecto, los elementos de una lista se muestran uno debajo del otro.

Para colocarlos horizontalmente se utiliza: 

display: flex;

Esto convierte la lista en un contenedor flexible y coloca automáticamente los elementos en fila.

Paso 4: Dar estilo a los enlaces

Los enlaces del menú pueden personalizarse utilizando CSS. 

.menu a {
    display: block;
    padding: 15px 20px;
    color: white;
    text-decoration: none;
}

La propiedad padding agrega espacio interno para que el menú sea más cómodo de usar.

La propiedad text-decoration: none elimina el subrayado de los enlaces.

Paso 5: Agregar un efecto hover

La pseudo-clase :hover permite cambiar el estilo de un elemento cuando el usuario pasa el cursor sobre él. 

.menu a:hover {
    background-color: #555;
}

En este caso, el color de fondo cambia cuando el cursor se posiciona sobre una opción del menú.

Menú desplegable

Una vez creado un menú básico, es posible agregar submenús desplegables para organizar mejor la navegación.

Un menú desplegable funciona mostrando nuevas opciones cuando el usuario pasa el cursor sobre un elemento del menú principal.

Para crear este efecto se utilizará la misma estructura explicada anteriormente, pero agregando una lista adicional dentro de una opción del menú.

¡Probalo en tu editor copiando el código! 
<nav class="menu-desplegable">

    <ul>

        <li>
            <a href="#">Inicio</a>
        </li>

        <li class="desplegable">

            <a href="#">Notas</a>

            <ul class="submenu">

                <li>
                    <a href="#">HTML</a>
                </li>

                <li>
                    <a href="#">CSS</a>
                </li>

                <li>
                    <a href="#">Git</a>
                </li>

            </ul>

        </li>

        <li>
            <a href="#">Contacto</a>
        </li>

    </ul>

</nav>

<style>

    .menu-desplegable ul {
        list-style: none;
        margin: 0;
        padding: 0;

        display: flex;

        background-color: #333;
    }

    .menu-desplegable li {
        position: relative;
    }

    .menu-desplegable a {
        display: block;

        padding: 15px 20px;

        color: white;

        text-decoration: none;
    }

    .menu-desplegable a:hover {
        background-color: #555;
    }

    .submenu {
        display: none;

        position: absolute;

        top: 100%;
        left: 0;

        min-width: 180px;

        background-color: #444;

        flex-direction: column;
    }

    .submenu li {
        width: 100%;
    }

    .desplegable:hover .submenu {
        display: flex;
    }

</style>

El resultado del menú desplegable anterior se muestra a continuación:

  • Inicio
  • Notas
    • HTML
    • CSS
    • Git
  • Contacto

Paso 1: Crear un submenú

Dentro de la opción Notas se agrega una nueva lista <ul>. 

<li class="desplegable">

    <a href="#">Notas</a>

    <ul class="submenu">

        <li>
            <a href="#">HTML</a>
        </li>

    </ul>

</li>

Esa segunda lista será el contenido desplegable.

Paso 2: Ocultar el submenú

El submenú debe permanecer oculto hasta que el usuario interactúe con el menú principal. 

.submenu {
    display: none;
}

La propiedad display: none hace que el elemento desaparezca completamente.

Paso 3: Posicionar el submenú

Para que el submenú aparezca debajo del elemento principal se utiliza posicionamiento CSS. 

.menu-desplegable li {
    position: relative;
}

.submenu {
    position: absolute;

    top: 100%;
    left: 0;
}

La propiedad position: relative convierte al elemento principal en una referencia de posición.

Luego, el submenú utiliza position: absolute para ubicarse respecto a esa referencia.

Paso 4: Mostrar el submenú con hover

Finalmente se utiliza la pseudo-clase :hover para detectar cuándo el usuario pasa el cursor sobre la opción principal. 

.desplegable:hover .submenu {
    display: flex;
}

Cuando esto ocurre, el submenú cambia de display: none a display: flex, haciendo que aparezca automáticamente.

Gracias a esta técnica es posible crear menús desplegables simples utilizando únicamente CSS.
