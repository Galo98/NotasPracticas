Introducción a HTML

¿Qué es HTML?

HyperText Markup Language (lenguaje de marcado de hipertexto) ó HTML sirve para modelar la estructura de una página web. Los elementos de HTML indican al navegador web cómo mostrar el contenido. Los elementos de HTML se definen con etiquetas. Los navegadores no muestran las etiquetas, sino que las usan para interpretar el contenido.

HTML puede ser codificado con cualquier editor de texto, para que el navegador lo reconozca, el archivo debe tener la extensión “.html”.

Algunos editores que nos ayudaran a codificar en HTML son SublimeText y VisualStudioCode. Estos editores ponen automáticamente color a nuestro código para guiarnos y muchas veces nos darán sugerencias al apretar una tecla o harán el cierre de etiquetas automáticamente. Para elegir uno de ellos es recomendable haberlos probado todos, incluyendo el block de notas.

Etiquetas HTML

Las etiquetas HTML generalmente se usan en pares una indica la aperturda y la otra el cierre. La etiqueta de apertura tiene el siguiente formato: 

<nombre_etiqueta_apertura> <\nombre_etiqueta_cierre>
El nombre de la etiqueta de cierre es igual al de la etiqueta de apertura pero con una barra (/) delante. Por ejemplo, para escribir un párrafo se usa la etiqueta: 
<p>
y para cerrarlo se usa: 
</p>
Algunas etiquetas no necesitan ser cerradas, como por ejemplo: 
<img>
que se usa para insertar imágenes.

La finalidad de un navegador web es leer las etiquetas HTML. El navegador no mostrará estas etiquetas; sino que las usará para determinar cómo tiene que mostrar el contenido. Puede haber diferencias entre los diferentes navegadores, aun así, la mayoría de los navegadores modernos siguen los mismos estándares de HTML, por lo que el contenido se mostrará de manera similar en todos ellos.

Caracteres especiales en HTML

Para escribir código HTML dentro de un documento HTML, es necesario usar las entidades HTML. Las entidades HTML son secuencias de caracteres que representan caracteres especiales. Por ejemplo, para escribir el símbolo menor que (<) se usa la entidad &lt;, para escribir el símbolo mayor que (>) se usa la entidad &gt; y para escribir el símbolo ampersand (&) se usa la entidad &amp;.

Tablas en HTML

Las tablas en HTML se crean con la etiqueta <table>. Cada fila de la tabla se define con la etiqueta <tr>. Las celdas de la tabla se definen con las etiquetas <td> para las celdas normales y <th> para las celdas de encabezado. A continuación se muestra un ejemplo de una tabla en HTML:

¡Probalo en tu editor copiando el código! 
<table>
    <tr>
        <th>Nombre</th>
        <th>Apellido</th>
        <th>Telefono</th>
    </tr>

    <tr>
        <td>Juan</td>
        <td>Navarro</td>
        <td>00000000</td>
    </tr>
</table>

El resultado de la tabla anterior se muestra a continuación:

Nombre Apellido Telefono
Juan Navarro 00000000
Milagros Romero 11111111

En la tabla anterior se muestra una tabla con dos filas y tres columnas. La primera fila es la fila de encabezado, que se define con la etiqueta <th>. La segunda fila es la fila de datos, que se define con la etiqueta <td>.

Para una mejor semántica de tablas, se pueden usar las etiquetas <thead>, <tbody> y <tfoot>. La etiqueta <thead> se usa para definir la sección de encabezado de la tabla, la etiqueta <tbody> se usa para definir la sección de cuerpo de la tabla y la etiqueta <tfoot> se usa para definir la sección de pie de la tabla. A continuación se muestra un ejemplo de una tabla con estas etiquetas:

¡Probalo en tu editor copiando el código! 
<table>
    <thead>
        <tr>  
            <th>Nombre</th>
            <th>Apellido</th>
            <th>Telefono</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td>Juan</td>
            <td>Navarro</td>
            <td>00000000</td>
        </tr>
        <tr>
            <td>Milagros</td>
            <td>Romero</td>
            <td>11111111</td>
        </tr>
    </tbody>
    <tfoot>
        <tr>
            <td>Total de registros</td>
            <td>&nbsp;</td>
            <td>2</td>
        </tr>
    </tfoot>
</table>

El resultado de la tabla anterior se muestra a continuación:

Nombre Apellido Telefono
Juan Navarro 00000000
Milagros Romero 11111111
Total de registros   2

En la tabla anterior se muestra una tabla con dos filas de datos y una fila de pie. La primera fila es la fila de encabezado, que se define con la etiqueta <th>. La segunda y tercera fila son filas de datos, que se definen con la etiqueta <td>. La cuarta fila es la fila de pie, que se define con la etiqueta <td>.

Se utiliza &nbsp; para dejar un espacio en blanco. Sin embargo, esta técnica no es recomendable para el diseño responsivo o para reemplazar un elemento vacío.
